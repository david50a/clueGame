# Clue Game

A Java implementation of the classic Clue (Cluedo) board game.

## About

This project was created on October 29, 2025, and implements the Clue board game in Java with game logic, player management, and detective gameplay.

## Languages

- Java

## Repository Info

- **Owner:** david50a
- **Visibility:** Public
- **Last Updated:** January 26, 2026

## Project Features

- Classic Clue/Cluedo game mechanics
- Player management
- Game board implementation
- Detective logic

## Prerequisites

- Java 8+
- IDE (IntelliJ, Eclipse, or VS Code with Java extensions)

## Building and Running

```bash
# Compile the Java files
javac -d bin src/**/*.java

# Run the game
java -cp bin ClueGame
```

## Game Rules

The game follows the standard Clue board game rules where players must deduce who committed the crime, where, and with what weapon.
